Occipital Structure SDK (cross-platform)  0.9
Linux

Prerequisites

The Structure SDK is built and tested on Ubuntu 16.04, but should work on other Linux distributions provided the following dependencies are available:

  • CMake 3.5+
  • GNU make 4.0+
  • gcc/g++ 5.3+
  • libusb 1.0+
  • OpenGL implementation providing libGL.so and libEGL.so
  • X11 libraries: libxcursor, libxi, libxinerama, libxrandr

The following packages may be installed on Ubuntu 16.04 to satisfy the above:

  • libc6-dev
  • libstdc++-5-dev
  • libgl1-mesa-dev
  • libegl1-mesa-dev
  • libusb-1.0-0-dev
  • libxcursor-dev
  • libxi-dev
  • libxinerama-dev
  • libxrandr-dev

The SDK is available for both x86_64 and arm64 (aarch64) architectures.

USB Device Permissions

To grant non-root users access to Structure Core devices, run the included udev rule installer:

$ sudo DriverAndFirmware/Linux/Install-CoreDriver-Udev-Linux.sh

To see usage information, pass --help on the command line.

Building

The SDK includes a shell script that automatically invokes the build tools:

$ Scripts/build.sh

A Makefile build system will be generated in builds/linux-CONFIG-ARCH. Pass --help to see available options.

Alternatively, run cmake and make directly:

$ mkdir build
$ cd build
$ cmake -G'Unix Makefiles' -DCMAKE_BUILD_TYPE=Release ..
$ make Samples

Once built, the sample apps may be run from the Apps directory.
