Occipital Structure SDK (cross-platform)0.9
Getting Started

Welcome to the Occipital cross-platform Structure SDK! This package contains:

  • The Structure library and headers
  • Sample apps for streaming data from Structure Core sensors
  • CoreFirmwareUpdater tool for all supported platforms

To build the sample apps, refer to the appropriate instructions for your platform:

  • Linux
  • Windows
  • macOS
  • Android

On each platform, the following build targets are defined:

  • CorePlayground: bare executable for the CorePlayground sample app
  • CorePlaygroundApp: generate CorePlayground app bundle (see below)
  • GuiTest: bare executable for the GuiTest sample app
  • GuiTestApp: generate GuiTest app bundle (see below)
  • SimpleStreamer: SimpleStreamer command-line tool
  • Samples: meta-target for all sample targets

The *App targets produce a platform-specific "app bundle".

  • Linux: Apps/AppName/AppName
  • Windows: Apps/AppName/AppName.exe
  • macOS: Apps/AppName.app
  • Android: Apps/AppName.apk
