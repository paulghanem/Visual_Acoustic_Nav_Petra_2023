Occipital Structure SDK (cross-platform)  0.9
macOS

Prerequisites

  • macOS 10.13+
  • CMake 3.5+
  • Xcode 10.0+ or equivalent Command Line Tools package

No driver installation is required.

Building

The SDK includes a shell script that automatically invokes the build tools:

$ Scripts/build.sh

A Makefile build system will be generated in Builds/macos-CONFIG-x86_64/. Pass --help to see available options.

Alternatively, invoke cmake and make directly:

$ mkdir build
$ cd build
$ cmake -G'Unix Makefiles' -DCMAKE_BUILD_TYPE=Release ..
$ make Samples

Or generate an Xcode build system:

$ mkdir build
$ cd build
$ cmake -G'Xcode' ..
$ open StructureSDK-CrossPlatform.xcodeproj
